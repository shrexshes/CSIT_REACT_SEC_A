PROMISES IN JS

Promises in js is an object that represents the eventual completion (or faliure) of an asynchronous operation and its resulting values. Its like a "promise" that something will happen and you can decide what to do when it does , or it fails. it helps to handle asynchronous task, like fetching from an api , reading a file or waiting for a timer. Promises have 3 states: 1. Pending : the task is still running 2. Fulfilled (Resolved): The task completed successfully 3. Rejected: The task failed.
