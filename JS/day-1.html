Variables

Variable are used to store data values. Variable types : var , let , const function scope : When a variable is declared inside a function , it is only accessible within that function and cannot be used outside that function. block scoped: A variable when declared inside the if or switch condition or inside for or while loops, are accessible within that particular condition or loop. var : Function scoped let and const block scoped

Reassigned and Constant

let : is for values you expect to change over time ( like a game score, a counter and a user status) const : stands for "constant". Once you assign it a value you cannot overwrite that value
